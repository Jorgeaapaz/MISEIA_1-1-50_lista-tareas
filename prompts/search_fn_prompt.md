# New Search Functionality

## Role
Act as a software developer, you are an expert in NextJS and MongoDB

## Task
Add a Search functionality on the main page, the search can be done by any of two options: search by `titulo` or by `completada`. Add the logic required to filter the list, if no matches are found, show a modal message with `No se han encotrado registros con los datos proporcionados` and include search data in the modal message.
